1. 视频(重点)

1.1 常见属性

controls  // 显示播放视频原生控件(兼容性差)
loop   // 循环播放
muted  // 静音播放
preload  // 预加载策略
none  // 不预加载任何数据
metadata  // 只预加载元数据
(视频长时/视频第一个画面/视频高度和宽度)
auto:加载元数据并且加载一定时长视频(默认)
autoplay 自动播放视频(大多数浏览器支持差)
poster:  在播放视频之前显示一张图片(广告)
如果视频暂停广告不再显示
volume    // 控制音量
