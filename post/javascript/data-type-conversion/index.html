数据类型的转换

· 隐式转换

问题一：自增（++）是否会发生隐式转换？

/* 前提先补充以下知识 */
var a;
a = 1;
a++; // a = a + 1, 结果为 2
a--; // a = a - 1, 结果为 0

a = '1';
a++; // 结果为 2, 原因是变量a的值'1'发生了隐式转换，由'1'转为1
typeof a; // 结果为 number

结论 1：变量发生自增时，字符串也会隐式转换为数字。

问题二：赋值运算符 += 是否也会发生隐式转换？

var a;
a = 2;
a += 2 // a = a + 2, 结果为 3

a = '2'
a += 1 // a = a + 2, 结果为 '21'
typeof a // 结果为 string

结论 2：赋值运算符 += 不会发生隐式转换。

· 强制转换

  1. parseInt

    计算公式：数字 * 进制数 ^ （第几个位数-1）

    例子：

   parseInt('123', 4) // 1*4^2 + 2*4^1 + 3*4^0 = 27
   parseInt('cf', 16) // 12*16^1 + 15*16^0 = 207
