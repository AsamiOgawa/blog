1. 数据类型

1.1 数据类型的分类

  • 基本(值)类型
    • string: 任意字符串
    • number: 任意数字
    • boolean: true和false
    • null: null
    • undefined: undefined
  • 对象(引用)类型
    • object
    • function: object中的一种特别类型, 特点是能够调用执行代码
    • array: object中的一种特别类型, 特点是值有小标(索引)

1.2 数据类型的判断

  • 判断方法: typeof 、instanceof、===

  • typeof : 返回字符串, 如

    var a;
console.log(typeof a);  // 'undefined'

  • instanceof : 返回boolean, 如

    var a = {
    a: '炒鸡蛋',
    b: 666,
    c: function () { },
    d: ['刘德华', null, 23]
}
console.log(a instanceof Object);	// true
console.log(a instanceof Array);	// false
console.log(a.d instanceof Array, a.d instanceof Object); // true, true(这里也说明了Array为Object, 只不过是一种特别的Object)

    a instanceof Object 意思: a 是否为 Object 的实例

    注意: A instanceof B, A找原型链(即__proto__指向的对象) ; B找显式原型属性对象(即为prototype属性对象)

​ 变量结束前, 通过赋值变量为null, 让存储在堆的对象成为垃圾, 让浏览器里的垃圾回收器回收垃圾对象

​ 数据类型: 基本类型、对象类型

​ 变量类型: 基本类型、引用类型

  • 数据类型: 基本类型、对象类型

  • 变量类型: 基本类型、引用类型

2. 函数

2.1 函数的定义

​ 函数是一个存储n条代码语句的封装体

2.2 函数的执行方式

  • 直接调用, 如 test()

  • 通过对象进行调用, 如 obj.test()

  • 通过new进行调用, 如 new test()

  • 通过 call/apply 的方式进行调用, 如 test.call(obj)

    注意: JavaScript的强大之处在于: 一个方法可以由任意的对象进行调用, 这是其他语言所没有的

2.3 回调函数

2.4 IIFE

  • 概念

​ IIFE是Immediately Invoked Function Expression的简写, 意思为立即调用函数表达式, 如以下例子:

(function () {
    var a = 123
    console.log(a);  // 123
})()

​ 所以IIFE也可以理解为匿名函数自调用

  • 作用: 隐藏代码(相对于全局来说), 防止变量污染, 做模块化

3.1 作用域的概念

  1. 什么是作用域？

    作用域就是代码区域, 但这个代码区域分为全局作用域和函数作用域

4. 原型

4.1 总结

1. 所有函数都是Function函数的实例对象
2. 除了Object函数的原型, 所有的函数的原型都是Object函数的实例对象
3. 原型链的尽头是Object函数原型的 `__proto__` ，值为null

5. 闭包

5.1 闭包的缺点

​ 造成内存泄漏，从而导致内存溢出

5.2 内存溢出与内存泄漏

​ 1. 什么是内存溢出？

​ * 内存溢出是程序运行的一种错误

​ * 当程序运行所需的内存超出了当前计算机所剩余的内存, 就会产生内存溢出

​ 2. 什么是内存泄漏？

​ * 内存泄漏是占用的内存没有及时释放

​ 3. 常见的内存泄漏

​ * 意外的全局变量, 比如在函数内部没有使用var 或 let 申明变量的关键字, 直接赋值从而导致全局变量的产生

​ * 未及时清除定时器或回调函数
